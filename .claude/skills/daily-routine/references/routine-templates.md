# routine-templates.md — 루틴 출력 템플릿

> 각 루틴 워크플로우가 수집한 데이터를 아래 템플릿에 채워서 최종 출력을 생성한다.
> `{변수명}`은 실행 시 실제 데이터로 치환된다.

---

## 1. 모닝 브리핑 템플릿

```markdown
## ☀️ 모닝 브리핑 — {YYYY-MM-DD} ({요일})

### 📋 어제 요약
{어제_데일리_로그가_없으면: "어제 기록이 없습니다."}
- {어제_활동_1}
- {어제_활동_2}
- {어제_활동_3}

### ☐ 오늘 할일 ({총_할일_수}건)
**🔴 긴급 (priority-high)**
{high_목록이_없으면: "없음"}
1. {할일_제목} — {프로젝트명}
2. ...

**🟡 중요 (priority-mid)**
{mid_목록이_없으면: "없음"}
1. {할일_제목} — {프로젝트명}
2. ...

**🟢 일반 (priority-low)**
{low_목록이_없으면: "없음"}
1. {할일_제목} — {프로젝트명}
2. ...

### 📅 오늘 일정
{일정이_없으면: "예정된 일정이 없습니다."}
- {시간} {일정_내용}
- {시간} {일정_내용}

### 📰 업계 동향
{뉴스_없으면: "뉴스 수집을 건너뛰었습니다."}
- **{키워드}**: {한_줄_요약} ([출처]({URL}))
- **{키워드}**: {한_줄_요약} ([출처]({URL}))
- **{키워드}**: {한_줄_요약} ([출처]({URL}))

### 💡 리마인더
{리마인더_없으면: "특별한 리마인더가 없습니다."}
- {메모리에서_가져온_중요_항목}

---
*생성: {HH:MM} | 할일 {총_할일_수}건 | 일정 {총_일정_수}건*
```

---

## 2. 저녁 리뷰 템플릿

```markdown
## 🌙 데일리 리뷰 — {YYYY-MM-DD} ({요일})

### ✅ 완료한 것 ({완료_수}건)
{완료_없으면: "오늘 완료한 항목이 없습니다."}
- ✅ {할일_제목} — {완료_시각}
- ✅ {할일_제목} — {완료_시각}

### ☐ 미완료 ({미완료_수}건)
{미완료_없으면: "모든 할일을 완료했습니다! 🎉"}
- ☐ {할일_제목} (importance: {N}) → {이월|보류|취소}
- ☐ {할일_제목} (importance: {N}) → {이월|보류|취소}

### ⚠️ 장기 미완료 (3일 이상)
{장기_미완료_없으면: "없음"}
- ⚠️ {할일_제목} — {생성일}부터 {N}일째 미완료

### 💡 오늘의 인사이트
{인사이트_없으면: "특별한 인사이트가 없습니다."}
- {데일리_로그에서_추출한_인사이트}

### 📊 생산성 스코어
**{완료_수} / {전체_수} = {퍼센트}%** {퍼센트>=80이면: "🔥", 50~79이면: "👍", <50이면: "💪"}

---
*리뷰 생성: {HH:MM} | 완료 {완료_수}건 | 이월 {이월_수}건*
```

## 3. 주간 회고 템플릿

```markdown
## 📊 주간 회고 — Week {주차} ({MM/DD} ~ {MM/DD})

### 프로젝트별 진행상황

**{프로젝트명_1}**
- 완료: {완료_항목_목록}
- 진행중: {진행중_항목_목록}
- 블로커: {있으면_기술, 없으면_"없음"}

**{프로젝트명_2}**
- 완료: {완료_항목_목록}
- 진행중: {진행중_항목_목록}
- 블로커: {있으면_기술, 없으면_"없음"}

{프로젝트가_없으면: "이번 주 프로젝트 활동이 기록되지 않았습니다."}

### 주요 의사결정
{의사결정_없으면: "이번 주 기록된 의사결정이 없습니다."}
1. **{결정_내용}** — {이유} ({날짜})
2. **{결정_내용}** — {이유} ({날짜})

### 주간 통계
| 항목 | 수치 |
|------|------|
| 총 할일 | {총_할일}건 |
| 완료 | {완료}건 ({퍼센트}%) |
| 이월 | {이월}건 |
| 새로 추가 | {신규}건 |
| 메모리 저장 | {저장}건 |

### 다음 주 우선순위
1. 🔴 {가장_중요한_것}
2. 🟡 {두_번째}
3. 🟢 {세_번째}

### 한 줄 회고
> {이번_주를_한_문장으로}

---
*회고 생성: {YYYY-MM-DD HH:MM} | {프로젝트_수}개 프로젝트 | {총_완료}건 완료*
```
